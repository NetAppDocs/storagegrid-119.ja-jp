= ストレージノードの管理
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ストレージ ノードは、ディスク ストレージ容量とサービスを提供します。ストレージ ノードの管理には次の作業が必要です。

* ストレージオプションの管理
* ストレージボリュームのウォーターマークとは何か、またウォーターマークのオーバーライドを使用してストレージノードが読み取り専用になるタイミングを制御する方法を理解する
* オブジェクトメタデータに使用されるスペースの監視と管理
* 保存されたオブジェクトのグローバル設定を構成する
* ストレージノード構成設定の適用
* フルストレージノードの管理
